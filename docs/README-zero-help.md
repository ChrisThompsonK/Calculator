# Calculator App

A simple web-based calculator application.

## Setup

```bash
# Install dependencies
npm install

# Start the application
npm run dev
```

## Usage

Open your browser to:
- http://localhost:3000
- http://localhost:3001

## Features

- Basic arithmetic operations
- Memory functions
- Calculation history
- Responsive design
